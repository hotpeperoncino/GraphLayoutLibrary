Detailed Description

Class GraphMLReader :- It is about implementing Graphml reader's functionality using default handler provided by xml parsing in qt with SAX approach.

Added 'GRAPHLAYOUTLIBRARYSHARED_EXPORT' before class name inorder to get access to Client. This ensures that the right macro is seen by both library and clients.
