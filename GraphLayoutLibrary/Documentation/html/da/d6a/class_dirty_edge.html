Detailed Description

DirtyEdge : This class is for maintaining the attributes of edges as we cannot directly access the attributes of edges in the endElement part of the Sax Handler So store the attributes of edge into member variables of this class using set() functiionality. and have to fetch those attributes in endElement using get() functionality.
